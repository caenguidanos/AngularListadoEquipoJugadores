## AngularListadoEquipoJugadores

#### 0. Create file _/backend/.env.local_ with `.example` values

### 1. MongoDb Runs on `MongoDb Atlas`

### 2. Run `backend`

- PORT: 3000

```bash
cd backend && \
    npm i && \
    npm run start:dev
```

### 3. Run `frontend`

- PORT: 4200

```bash
cd frontend && \
    npm i && \
    npm start
```
